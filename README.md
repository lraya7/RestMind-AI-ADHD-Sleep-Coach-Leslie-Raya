**Project Description: RestMind AI**

RestMind AI is an AI-powered ADHD sleep coach designed to help people calm racing thoughts and build a healthier bedtime routine. The app supports users by letting them complete a “brain dump,” organize their worries or tasks, and receive personalized calming strategies such as breathing exercises, wind-down routines, or sleep stories. The goal is not to replace medical care, but to give users a gentle tool that helps them feel less overwhelmed at night.

This project connects to AI because it uses natural language processing to understand what the user is feeling, classifies the type of sleep problem, recommends a personalized response, and improves over time based on user feedback. For example, if a user writes, “My brain won’t stop thinking about tomorrow,” the AI can detect racing thoughts or task anxiety and suggest a short brain dump followed by a calming exercise. Over time, RestMind AI can learn which strategies work best for each user and create a more personalized sleep routine.
