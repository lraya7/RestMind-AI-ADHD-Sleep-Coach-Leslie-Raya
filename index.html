RestMind AI

An ADHD sleep coach that turns racing thoughts into a calming bedtime plan.

View Demo Plan How It Works

Created by Leslie Raya · Computer Science

The Problem

Bedtime can feel like the brain is finally “wide awake.”

Racing thoughts

Unfinished tasks, worries, and reminders often show up right when the user is trying to rest.

Dopamine loop

Phone scrolling can become a quick dopamine cycle that makes it harder to begin a wind-down routine.

No structure

A routine can be difficult to start when the user feels overwhelmed or does not know what to do first.

Generic advice

Basic sleep tips do not always adapt to the user’s mood, energy, ADHD patterns, or current obstacle.

“I forgot...” “What if...” “One more video...”

Project Goal

A supportive AI assistant that helps users prepare for sleep.

RestMind AI is a wellness support tool. It does not diagnose, treat, or replace professional care.

1

Empty the mind

A fast brain dump turns random bedtime thoughts into organized tomorrow tasks, worries, or reminders.

2

Choose a calming action

The AI recommends a short, realistic strategy based on the user’s current mood and energy level.

3

Build a routine

Over time, the assistant learns which wind-down steps help the user fall asleep faster.

Design promise: gentle, short, personalized, and nonjudgmental.

How the AI Model Works

Input → understand the sleep problem → recommend a calming response → learn from feedback.

User Input

“My brain won’t stop.”

NLP Classifier

Detects racing thoughts, stress, overstimulation, or restlessness.

Recommendation Engine

Chooses breathing, brain dump, sleep story, or routine support.

Feedback Loop

Learns what helped using bedtime, wake time, mood, and rating.

AI Techniques

  • Text classification for sleep-state detection
  • Prompted LLM responses for calming guidance and stories
  • Recommendation rules based on mood, energy, and history
  • Optional Graph RAG to connect routines, triggers, and strategies

Tech Stack

Frontend: React or HTML/CSS Backend: Python + FastAPI Model: OpenAI / Groq / Local LLM Data: JSON or SQLite Deploy: GitHub Pages + Render

Core Features

One prompt, one action, one next step.

🌙

Brain Dump Organizer

Sorts thoughts into tomorrow tasks, worries, reminders, and “let go for tonight.”

🫁

Wind-Down Coach

Ethics + Safety

Because sleep and ADHD connect to mental wellness, the app must be careful.

Not medical diagnosis

The assistant clearly states that it is a wellness support tool, not a doctor or therapist.

Privacy first

Sleep notes may include personal worries, so data should be stored securely and minimized.

Crisis-aware limits

If the user mentions self-harm or severe distress, the app should provide urgent support resources.

No-shame design

Language avoids guilt and gently encourages restarting after difficult nights.

Low stimulation

The interface avoids bright visuals, endless chats, or exciting stories that keep users awake.

Final Vision

RestMind AI is not just a sleep app. It is a supportive AI companion that helps people with ADHD calm their thoughts, build healthier routines, and feel more in control at night.
